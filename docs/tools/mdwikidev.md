mdwiki のソースコードから自分でビルドしてみる。

## 必要なツール

- Git
- node.js / nvm

## 手順

1. [Github](https://github.com/Dynalon/mdwiki)から、mdwiki のリポジトリを Clone する
2. make ファイルを実行する。
3. dist フォルダができて、そこに mdwiki.html がある。

「Windows の場合は、make ファイルの中のコマンドを手動実行」と書いてあるけど、Bash for Windows を使った方が楽だった。
